JavaScript Syntax

JavaScript syntax is the set of rules, how JavaScript programs are constructed:

  • JavaScript is case-sensitive
  • JavaScript statements are separated by semicolon
  • JavaScript blocks are enclosed in curly brackets
  • JavaScript is written in HTML

JavaScript Variables

JavaScript variables are containers for storing data values.
JavaScript variables are declared with the var keyword:

        var x;
        x = 5;
        var y = 6;
        var z = x + y;

        // Menggunakan var (tidak disarankan untuk kode modern)
        var nama = "Ahmad Iffat";
        var umur = 21;

        // Menggunakan let (disarankan untuk kode modern)
        let kota = "Makassar";
        let isStudent = true;

        // Menggunakan const (untuk nilai yang tidak akan berubah)
        const tahunLahir = 2000;
        const PI = 3.14;

JavaScript Data Types

JavaScript variables can hold many data types: numbers, strings, objects and more.<>JavaScript has dynamic types. This means that the same variable can be used to hold different data types:

        var x;           // Now x is undefined
        x = 5;           // Now x is a Number
        x = "John";      // Now x is a String

JavaScript Operators

JavaScript uses arithmetic operators to compute values (just like algebra).

JavaScript uses an assignment operator to assign values to variables.

JavaScript uses comparison operators to compare values.

JavaScript uses logical operators to determine the logic between variables or values.

        // Arithmetic operators
        var x = 5;
        var y = 2;
        var z = x + y;   // Addition
        var z = x - y;   // Subtraction
        var z = x * y;   // Multiplication
        var z = x / y;   // Division
        var z = x % y;   // Modulus
        var z = x ** y;  // Exponentiation

        // Assignment operators
        var z = x++;     // Increment
        var z = x--;     // Decrement
        var z = x += y;  // Assignment

        // Comparison operators
        var z = x == y;  // Equal to
        var z = x === y; // Identical
        var z = x != y;  // Not equal
        var z = x !== y; // Not identical
        var z = x > y;   // Greater than
        var z = x < y;   // Less than
        var z = x >= y;  // Greater than or equal to
        var z = x <= y;  // Less than or equal to
        
        // Logical operators
        var z = x && y;  // Logical and
        var z = x || y;  // Logical or
        var z = ! x;     // Logical not
    
        // Contoh operator aritmatika
        let x = 10;
        let y = 5;
        let sum = x + y; // Penjumlahan
        let diff = x - y; // Pengurangan

        // Contoh operator perbandingan
        let isEqual = (x == y); // false, karena x tidak sama dengan y

        // Contoh operator logika
        let isAdult = (age >= 18) && (age < 60); // true, jika age lebih dari 18 dan kurang dari 60

JavaScript Condition

Conditional statements are used to perform different actions based on different conditions.

  • Use if to specify a block of code to be executed, if a specified condition is true.
  • Use else to specify a block of code to be executed, if the same condition is false.
  • Use else if to specify a new condition to test, if the first condition is false.
        let score = 85;
        if (score >= 90) {
            console.log("Grade A");
        } else if (score >= 75) {
            console.log("Grade B");
        } else {
            console.log("Grade C");
        }

JavaScript Loop

Loops can execute a block of code a number of times.

Loops are handy because they save time, reduce errors, and they make code more readable.

  • for - loops through a block of code a number of times
  • for/in - loops through the properties of an object
  • for/of - loops through the values of an iterable object
  • while - loops through a block of code while a specified condition is true
  • do/while - also loops through a block of code while a specified condition is true
        // Perulangan while
        let i = 0;
        while (i < 5) {
            console.log("i is: " + i);
            i++;
        }

        // Perulangan for
        for (let i = 0; i < 5; i++) {
            console.log("i is: " + i);
        }

JavaScript Array

JavaScript arrays are used to store multiple values in a single variable.

An array is a special variable, which can hold more than one value at a time.

        // Membuat array
        let cars = ["Toyota", "Honda", "Suzuki"];

        // Mengakses elemen array
        let myCar = cars[0]; // Output: Toyota

        // Mengubah elemen array
        cars[0] = "Daihatsu";

        // Menambah elemen array
        cars.push("Mitsubishi"); // Menambahkan elemen di akhir array

        // Mengetahui panjang array
        let length = cars.length; // Output: 4

JavaScript Functions

A JavaScript function is a block of code designed to perform a particular task.

A JavaScript function is executed when "something" invokes it (calls it).

        function myFunction(p1, p2) {
            return p1 * p2;   // The function returns the product of p1 and p2
        }

        // Contoh
        fuction salam(nama) {
            return "Halo, " + nama + "!";
        }

        // Memanggil fungsi
        let message = salam("Ahmad Iffat");

        // Menampilkan pesan
        console.log(message);
    
        // Function declaration
        function greet() {
            console.log("Hello!");
        }

        // Function expression
        let greet = function() {
            console.log("Hello!");
        }

        // Arrow function
        let greet = () => {
            console.log("Hello!");
        }

            greet(); // Output: Hello!

        // Arrow function (shorter)
        let greet = name => console.log("Hello! " + name);

            greet("Ahmad Iffat"); // Output: Hello! Ahmad Iffat

        // Return Value Function
        function sum(a, b) {
            return a + b;
        }

            let result = sum(5, 3);
            console.log(result); // Output: 8
        
        // Function with default parameter
        function greet(name = "Stranger") {
            console.log("Hello! " + name);
        }

            greet(); // Output: Hello! Stranger
            greet("Ahmad Iffat"); // Output: Hello! Ahmad Iffat
        
        // Function with rest parameter
        function sum(...numbers) {
            let result = 0;
            for (let number of numbers) {
                result += number;
            }
            return result;
        }

            let result = sum(1, 2, 3, 4, 5);
            console.log(result); // Output: 15

        // Anonymous function
        let greet = function(name) {
            console.log("Hello, " + name + "!");
        }

            greet(); // Output: Hello!
